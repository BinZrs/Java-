## 实验5-2 获取系统的文件树

**实验内容：**

  1. 获取某个目录下的目录信息，目录从控制台输入。
  2. 目录信息包括该目录下的所有文件和文件夹的列表。对于每个文件夹显示其名称，修改日期。对于每个文件显示其名称，修改日期和大小。
  3. 目录的信息写出到文件中。注意，文件信息首先按类型分类，文件夹在前，文件在后，并且他们各自按照文件名字符串的顺序排序。
  4. 扩展以及自学部分：利用JAVA的GUI（Swing等）创建一个简易的文件浏览器。
